# Roadmap

This project is in early development. The goal is to provide practical quality checks for PLC/HMI/SCADA tag lists used in industrial automation projects.

## Version 0.1.0

- CSV tag list validation
- Duplicate tag name detection
- Duplicate address detection
- Missing description detection
- Basic data type validation
- Basic naming convention check
- Markdown report generation

## Planned features

- Excel file support
- Siemens TIA Portal tag export support
- WinCC tag list validation
- SCADA tag dictionary validation
- JSON report output
- Custom naming convention profiles
- CI integration for automation project repositories
- More detailed industrial automation validation rules

## Long-term idea

The long-term goal is to create a small open-source toolkit for improving documentation quality, diagnostics, and maintainability of PLC/HMI/SCADA projects.
